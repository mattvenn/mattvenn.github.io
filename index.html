Multi Project Harness

I’ve been working on the multi project harness. Normally in a multi project wafer you would have the entire space for your own design. The Google/Skywater Shuttle is a bit different. First of all, there is the Caravel harness, which even has its own datasheet For this course, I want to put all your designs together into that area, so we need to do some extra bits: hold all the designs except 1 in reset connect all the inputs and outputs make sure the inputs and outputs don’t clash with each other connect important signals like clocks and make sure the tools know they are special Then there will need to be a bit of firmware on the SoC that sets up the GPIOs for each design.
