Inverter

A standard cell is a building block that contains some basic digital or analogue functionality. These blocks are then tiled together to make the final design. Since finding out about the Skywater PDK, I wanted to try drawing my own ‘standard cell’ using the Skywater transistor models. An inverter is one of the simplest and even that was quite difficult!
